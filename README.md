# DLsite-Classification

高效能 DLsite 分類工具
DLsite-Classification 是一款功能強大且高效的工具，專為自動化 DLsite 內容分類與管理而設計。不論是管理大量作品還是需要精確分類，這款工具都能提供全面的解決方案。

## 主要功能

- 非同步操作：使用 async 技術進行高速資料爬取與檔案處理。
- 自動提取作品代碼：自動識別並提取作品代碼（如 RJ, BJ, VJ, RE, BE, VE），實現無縫分類。
- 增強的元數據管理：自動為作品添加圖片、標籤和訊息，便於組織。
- 資料夾提取：深度掃描並提取資料夾內容，智能分類。
- 重複檢測：掃描並回報重複的檔案或資料夾，保持資料庫整潔。
- 資料夾比對：比對資料夾以識別差異或重複項目。
- API 與前端支援 (開發中)：提供 API 與使用者友好的前端介面，用於管理與查看作品資訊。

## 為什麼選擇 DLsite-Classification？

- 高效：透過自動化流程與非同步操作節省時間。
- 精確：使用先進的提取與比對工具，確保分類準確。
- 可擴展性：輕鬆處理大量作品，功能強大。
- 使用者友好：設計簡單易用，即使是初學者也能快速上手。
